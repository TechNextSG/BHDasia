Project History

BHD Asia — Website Mockup

A complete record of all versions, changes, and improvements made to the BHD Asia static website mockup. The site is a faithful HTML/CSS replica of bhdasia.com, built as a multi-page static site and hosted on GitHub Pages.

Live site: technextsg.github.io/BHDasia  ·  Repository: github.com/TechNextSG/BHDasia

v3.0 11 June 2026

Corporate Redesign

Design System

  • CSS custom properties introduced — --navy, --blue, --off-white, --text, --body, --muted, --shadow, --r, --t — one place to change the entire palette
  • Header changed to always-solid white with bottom border and scroll-triggered shadow — proper corporate standard, removing transparent/scroll-toggle behaviour
  • Dark sections now use deep navy #0b2238 instead of plain grey #2b2b2b — more premium feel
  • Section labels added — small blue uppercase text above all major section headings for consistent visual hierarchy
  • Buttons redesigned — solid blue fill as primary, plus btn-outline, btn-white, and btn-white-outline variants
  • Cards upgraded — box-shadow depth with hover lift effect replaces flat border treatment
  • Divider lines — short 48px blue horizontal rules added under section headings

Home Page

  • Hero — gradient overlay (navy to transparent) replaces flat black; animated scroll indicator added; hero-label, large headline, subtitle, and two CTA buttons
  • Three Pillar section added — Organisation Development, Resilience Building, Individual Development — hover cards with border-top accent
  • Stats bar added — deep navy strip displaying 27+ years, 1,300+ coaching hours, 9 countries, and certified since 2012
  • Voice record (Noomii widget) kept and given a dedicated Featured Interview section with supporting copy
  • Forbes Coaches Council section added — linked media highlight with Isabelle's photo and external link
  • CTA strip — discovery call + upcoming events CTAs

All Pages

  • All page-specific styles updated to use CSS variables
  • Event cards — navy header background, blue left border, shadow on hover
  • Service cards — bordered-top blue, box shadow, consistent grid layout
  • Partner cards — hover lift, shadow depth, tighter typography
  • Footer widgets — deeper navy, headings with bottom rule separator
  • About page — section labels, divider lines, mission and values headings updated

Design Polish & Content Summarisation

Design Improvements

  • Partners page — rebuilt from stacked full-width cards to compact 2-column grid; border-top accent added; descriptions tightened to 1–2 sentences per partner

  • Contact page — emoji icons (📍📞✉🏢) replaced with clean CSS symbol tiles; typography hierarchy tightened

  • Shared CSS utilities added to style.css — .page-intro, .client-tags, .dark-callout, .cta-strip

v1.0 11 June 2026

Initial Build — Full Multi-Page Mockup

New Files Created

  • style.css — shared stylesheet for all pages; Open Sans font, container layout (max-width 1080px), sticky header, Divi button styles, 4-column footer, responsive breakpoints
  • shared.js — sticky header scroll behaviour and mobile navigation toggle

  • index.html — home page with parallax hero, Noomii voice interview widget (Featured section)
  • about.html — Our Story, full client list, 3-column team section (Isabelle, Rouviere, Clement), Mission and Values
  • events.html — TRE® workshop listings with pricing tables (Super Early Bird / Early Bird / Normal tiers) for Aug and Sep 2026

Design Reference

  • Site built as a faithful static HTML replica of bhdasia.com (WordPress + Divi theme v4.7.7)
  • Divi class naming conventions preserved — et_pb_section, et_pb_row, et_pb_column_*, et_pb_fullwidth_header, et_pb_button
  • Primary accent colour #2ea3f2 (Divi blue), body text #666, headings #333, footer background #2b2b2b
  • Font: Open Sans (Divi default), weights 300–800
  • All team photos and logo loaded from bhdasia.com CDN with fallbacks
